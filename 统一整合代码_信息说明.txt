                    无人机运输系统统一分析程序 - 信息说明

一、系统概述
================================================================================
本程序整合了三个核心模块：
1. 地形3D渲染模块 - 基于PyVista实现高精度地形和建筑物可视化
2. HDBSCAN聚类模块 - 基于建筑物/POI数据自动识别无人机起降点
3. 无人机路径规划模块 - 计算运输时间、电量消耗等关键指标

三、聚类算法参数
================================================================================
1. HDBSCAN参数
   - MIN_CLUSTER_SIZE: 15 (最小簇大小)
   - MIN_SAMPLES: 3 (密度估计参数)
   - METRIC: euclidean (欧几里得距离)

2. 聚类策略
   - 使用人口密度加权进行特征标准化
   - POP_DENSITY_WEIGHT: 2.0 (人口密度权重)
   - CENTER_WEIGHT: 0.6 (密度中心权重)
   - DENSITY_WEIGHT: 0.4 (人口密度权重)

3. 起降点命名规则
   - 按字母顺序命名: A, B, C, D, ...
   - 每个字母对应唯一的地理坐标

四、路径规划模型
================================================================================
1. 代价函数
   总代价 = 距离代价 + 高度惩罚 + 转向惩罚 + 障碍物距离惩罚

2. 中国无人机飞行法规（根据2025年《无人驾驶航空器飞行管理暂行条例》）
   - 轻型无人机（250g-4kg）：真高≤120米
   - 微型无人机（≤250g）：真高≤50米
   - 真高=无人机到正下方实时地表的垂直高度（非海拔高度）
   - 本系统使用DJI Matrice 300 RTK（属于轻型无人机），限制真高120米

3. 惩罚参数（根据法规和用户要求设定）
   ┌─────────────────────────────────────────────────────────────────┐
   │  惩罚类型        │  软限制        │  硬限制        │  惩罚系数  │
   ├─────────────────────────────────────────────────────────────────┤
   │  高度惩罚        │  >60m开始惩罚  │  >120m严格限制│  50.0      │
   │  转向惩罚        │  转角越大惩罚越高（通过角度计算）           │  30.0      │
   │  障碍物距离惩罚  │  <10m开始惩罚  │  接触障碍物   │  100.0     │
   └─────────────────────────────────────────────────────────────────┘

4. 路径优化策略
   - 使用KD树加速障碍物查询
   - 绕障高度: 建筑物高度 + 20m安全距离
   - 最小转弯半径: 根据速度和高度自动计算
   - 代价计算: 总代价 = 高度代价 + 转向代价 + 障碍物代价

4. 3D距离计算
   - 使用欧几里得公式计算三维空间距离
   - 考虑地形起伏和建筑物高度

五、3D渲染参数
================================================================================
1. 地形渲染
   - 网格分辨率: 200×200
   - 颜色映射方案（按高度分级）:
     * 低海拔 (<20%): 深绿色 [0.25, 0.55, 0.28]
     * 中低海拔 (20-40%): 浅绿色 [0.35, 0.68, 0.35]
     * 中海拔 (40-60%): 黄绿色 [0.50, 0.65, 0.30]
     * 中高海拔 (60-80%): 棕黄色 [0.65, 0.58, 0.25]
     * 高海拔 (>80%): 浅棕色 [0.80, 0.75, 0.55]
   - 不透明度: 0.95

2. 建筑物渲染
   - 建筑物颜色（按高度分级，高度差异更明显）:
     * 0-30m:  深绿色 [0.20, 0.45, 0.20]  （低矮建筑）
     * 30-60m: 浅绿色 [0.30, 0.65, 0.35]  （中小建筑）
     * 60-100m: 黄绿色 [0.50, 0.75, 0.40] （中等建筑）
     * 100-150m:棕黄色 [0.75, 0.60, 0.20] （高层建筑）
     * >150m:   深红色 [0.90, 0.15, 0.10] （超高层建筑）
   - 圆柱体分辨率: 8 (优化性能)
   - 不透明度: 0.90

3. 起降点标记
   - 标记形状: 球体
   - 球体半径: 12m
   - 颜色: 红色
   - 标签高度偏移: +25m

4. 运输路线渲染
   - 路线颜色: 蓝色
   - 路线线宽: 2
   - 路线不透明度: 0.6
   - 分段数量: 根据距离动态调整

七、输出指标说明
================================================================================
1. 直线距离 (m): 两点间的二维平面距离
2. 实际距离 (m): 考虑地形起伏的三维实际飞行距离
3. 飞行时间 (分钟): 实际距离 / 飞行速度
4. 耗电量 (%): (飞行功率 × 飞行时间) / 电池容量 × 100%

八、性能优化
================================================================================
1. Numba JIT编译加速
   - 3D距离计算函数使用@jit装饰
   - 路径代价计算使用并行计算 @jit(parallel=True)

2. 批量渲染优化
   - 建筑物使用MultiBlock批量添加
   - 降低圆柱体分辨率(8)以提升渲染速度

3. K近邻加速
   - 仅检测最近的k个障碍物
   - 大幅减少碰撞检测计算量

九、操作说明
================================================================================
1. 运行程序
   python 统一整合代码.py

2. 3D可视化操作
   - 左键拖动: 旋转视角
   - 右键拖动: 缩放
   - 滚轮: 缩放
   - 中键拖动: 平移
   - 关闭窗口: 程序结束

3. 调节参数
   - MIN_CLUSTER_SIZE: 增大则减少起降点，减小则增加起降点
   - UAV_SAFE_HEIGHT_M: 调整安全飞行高度
   - 颜色定义: 在相应配置区域修改RGB值

十、注意事项
================================================================================
1. 建筑物必须位于地形上方，不得嵌入地下
2. 聚类点需在地图范围内，否则会自动裁剪
3. 耗电量超过100%表示该路线超出单次飞行范围
4. 碰撞检测不考虑路径平滑优化
5. 实际飞行前需进行现场勘查确认安全
